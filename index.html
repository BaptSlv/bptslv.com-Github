StarTDL

De quoi s'agit-il ?

StarTDL, c'est un projet personnel libre d'accès à chaque personne souhaitant en savoir plus sur mon parcours et mes compétences en terme de développement web. Site internet réalisé intégrallement par mes soins, j'ai voulu mettre en évidance les différentes technologies que je maitrise dans un projet concret. Le front end du site a été réalisé via Bootstrap et le back end est développé en PHP. Vous pouvez trouver le code source sur mon github.

Bonne découverte.

Gestion de to do list

Une partie du site TDLStar est consacrée à un projet de création de To Do List. Ajoutez, modifiez, supprimez, vous pouvez mettre à jour votre To Do List quand vous le souhaitez. Pour se faire, enregistrez-vous sur le site, connectez vous, et créez votre première liste !

Une offre premium fictive disponible

Il est tout à fait possible de tester l'intégralité des fonctionnalités du site TDLStar gratuitement. Cependant, la gestion d'un panier est une fonctionnalité majeure pour de nombreux sites internets. Il vous est donc possible de découvrir notre catalogue d'offres fictives afin de tester la gestion de panier mise en place sur ce site. La partie paiement n'a pas été ajoutée puisque les offres ne sont pas réellement disponibles.
